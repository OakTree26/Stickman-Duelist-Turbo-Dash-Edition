# Stickman-Duelist-Turbo-Dash-Edition
Stickman Duelist is a local 2 player fighting game that can be played on a single computer with gamepad support. pick up different weapons which falls from the sky. you can dash, block, jump, crouch, heal from medi kits and use different weapons with different advantages. it has scrolling background and camera.

# featurs
* Local 2-player game
* Random weapon drops (Sword, Katana, Knife, Steel Rod)
* Med Kits to restore health
* Dash mechanic
* Blocking system
* Jump and crouch
* Combo counter
* Health & stamina system
* Dynamic camera that follows both players
* Screen shake effects
* Particle effects and floating damage numbers
* CTR visual style
* Scrolling environment with platform
* Keyboard and gamepad support
* Sound effects generated using the Web Audio API

# Challenges
### Some of the biggest problems I had while making this game were:
* Fixing weapon collision detection
* Making every weapon feel different
* Preventing players from leaving the map boundaries
* Creating a smooth camera that follows both players
* Balancing stamina usage for attacking and dashing
* Adding gamepad support alongside keyboard controls

# Development
### The game started as a simple fighting prototype.
Over time I added:
* Weapon spawning system
* Pickup mechanics
* Dash movement
* Blocking
* Combo system
* Health and stamina bars
* Screen shake
* Dynamic camera
* Better animations
* Particle effects
* Multiple weapons and healing item

# AI usage
* This project was primarily built and developed by me.
* I used AI as a learning and debugging assistant to help fix JavaScript errors, understand some programming concepts and improve small parts of the code. The gameplay mechanincs, balancing, testing, feature additions and overall game design were implemented and modified by me throughout development.
